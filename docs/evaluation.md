## Detailed Scenario Notes

1. **AC-009 Delay Investigation (EVAL-001)**: Asserted the full set of 11 evidence and dependency IDs. Verified the synthesized milestone dates match the database: planned milestone date `2026-06-29`, forecast date `2026-07-05`, resulting in exactly a `6` day delay.

Public error messages do not leak internal structures, database schemas, or policy detail codes.

The assertions verify `AC-008` is `amber` and `AC-009` is `red`.
